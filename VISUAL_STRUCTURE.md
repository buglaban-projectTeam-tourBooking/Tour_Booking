## 📱 Responsive Breakpoints

### Desktop View (1200px+)
```
Full-width layout, 4-column grids, all content visible
```

### Tablet View (768px - 1199px)
```
Container padding reduced, 2-column grids in places, 
adjusted spacing for tablets
```

### Mobile View (< 768px)
```
Single column layout, stacked cards, adjusted hero, 
mobile-friendly navigation
```

## 📊 Spacing Guidelines

```
Large spacing: 30-50px (hero, sections)
Medium spacing: 20-30px (between blocks)
Small spacing: 10-15px (within components)
Card padding: 15-20px
Button padding: 8-12px (height)
```

## ⚡ Animations

```
Hover Effects:
- Cards: translateY(-5px) to (-8px)
- Buttons: backgroundColor change + transform
- Links: color change + padding shift

Transitions:
- Duration: 0.3s
- Timing: ease

Scroll Behavior:
- Smooth scroll enabled
- Back-to-top: fade in/out on scroll 300px
```

## ✨ Design Consistency

- **Typography:** All headings bold, consistent sizing
- **Spacing:** Uniform padding/margins
- **Colors:** Limited palette (5-6 colors)
- **Shadows:** Consistent box-shadow patterns
- **Borders:** Minimal, subtle where used
- **Radius:** Consistent border-radius (4-8px)
- **Transitions:** All 0.3s ease
- **Responsive:** All sections stack properly
